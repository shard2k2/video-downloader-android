# Video Downloader for Android

An Android app to download videos and audio from YouTube, Facebook, and other supported platforms. Built with Kotlin using yt-dlp and ffmpeg binaries.

## Features

- Paste any YouTube or Facebook URL to download
- Choose quality: 1080p, low quality, or MP3 audio
- MP3 downloads include metadata — title, artist name, and thumbnail
- Download queue — multiple downloads run one after another
- Cancel any download mid-progress
- Share directly from YouTube or Facebook app to trigger automatic download
- Live progress notifications with cancel button
- Completion notification — tap to open the downloaded file
- Downloads tab showing active, completed, and cancelled downloads

## Built With

- Kotlin
- yt-dlp
- ffmpeg

## Notes

Currently optimized for Samsung devices. Background process handling may vary on Xiaomi/MIUI devices due to aggressive battery management.
